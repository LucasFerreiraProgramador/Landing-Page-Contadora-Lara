# 🏢 Lara Luisa | Assessoria Contábil Estratégica

Uma Landing Page de alta conversão desenvolvida para um escritório de contabilidade. O projeto foi projetado com foco em UI/UX para transmitir confiança, autoridade e facilitar a captação de clientes através de chamadas de ação (CTAs) estratégicas.

## 🎯 Foco do Projeto e Características

Diferente de sistemas internos, esta aplicação foi estruturada para ser a "vitrine digital" da profissional. Os principais destaques incluem:

- 🚀 **Otimização para Conversão:** Botões de CTA bem posicionados e um botão flutuante do WhatsApp para contato rápido.
- 📱 **Mobile-First & Responsividade:** Layout fluido que se adapta perfeitamente a smartphones, tablets e monitores ultrawide.
- 🎨 **Identidade Visual Premium:** Uso de psicologia das cores (Azul Marinho e Dourado) para transmitir segurança, transparência e profissionalismo.
- ✉️ **Seção de Captação de Leads:** Formulário de contato dinâmico e seção de agendamento de consultas.
- 🧩 **Arquitetura de Informação:** Seções bem divididas (Hero, Sobre a Profissional, Serviços Detalhados e Rodapé Institucional com validação de CRC/CNPJ).

## 💻 Como Executar Localmente

Se você deseja clonar e rodar este projeto na sua máquina para fins de estudo:

### Pré-requisitos
- Node.js 18+ instalado
- Gerenciador de pacotes (npm, yarn ou pnpm)

### Instalação

```bash
# Clone o repositório
git clone https://github.com/LucasFerreiraProgramador/Landing-Page-Contadora-Lara

# Entre no diretório
cd Landing-Page-Contadora-Lara

# Instale as dependências
npm install
```
